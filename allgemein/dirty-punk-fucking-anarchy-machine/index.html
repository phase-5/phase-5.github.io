Dirty Punk Fucking Anarchy Machine

Die Dirty Punk Fucking Anarchy Machine ist quasi die Weiterentwicklung von "Rock Band" für die Playstation 3, nur älter. Macht aber immer noch Spaß. Man kann u.a.harmonisches Rülpsen einfügen.

Und was haben die Punks, eigentlich, dass sie einerseits auf Anarchie abfahren und dann wieder die Europäische Union abfeiern (Eu! Eu! Eu!)??? Entschuldigung, das musste...

Dies ist ein Gastbeitrag von Klaus.
