“Dean’s PermaLink Migration” in der Version 1.1 ist draußen

Damit kann man seine Linkstruktur ändern, wobei die alten Links auf die neuen umgeleitet werden. Sehr praktische Angelegenheit. Allerdings haben sich jetzt Sicherheitslücken offenbart, so dass eine Aktualisierung auf Version 1.1 anzuraten ist. Diese steht hier zum Download.
