I just wanted to point out two things to you that could've saved you some trouble.

1. Parallels Desktop 3.0 enables you to use the Bootcamp partition as the hard drive for your Parallels virtual machine. Hence you could have been using the exact same Vista install as the one that you're booting into using Bootcamp.

2. Windows Vista's license agreement explicitly forbids the use of Vista inside virtual machines, if I recall correctly. Microsoft does believe that a virtual machine is a seperate system and thus you'll need either a seperate license or a special one.
